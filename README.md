# Sitio Web HTML5

Proyecto desarrollado utilizando HTML5 con enfoque en estructura semántica, organización profesional del contenido y buenas prácticas de desarrollo web.

## 📌 Descripción

Este proyecto tiene como objetivo demostrar el uso correcto de etiquetas semánticas modernas en HTML5, así como una estructura clara y organizada del DOM. 

Se implementa una jerarquía adecuada de encabezados, secciones, artículos y footer para representar correctamente la información en un sitio web básico pero estructurado profesionalmente.

## 🛠 Tecnologías utilizadas

- HTML5
- Estructura semántica
- Buenas prácticas de organización del código

## 🎯 Objetivos del proyecto

- Aplicar etiquetas semánticas como:
  - `<header>`
  - `<nav>`
  - `<main>`
  - `<section>`
  - `<article>`
  - `<footer>`
- Mantener una estructura limpia y legible
- Preparar la base para futuros estilos con CSS
- Comprender la organización correcta del DOM

## 🚀 Posibles mejoras futuras

- Integración con CSS para diseño responsivo
- Implementación de JavaScript para interactividad
- Optimización para accesibilidad
